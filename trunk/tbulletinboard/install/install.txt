Installatie
-----------
1. Kopieer de php files naar een omgeving waarin PHP 5 draait.
2. Bewerk de config/settings.php file naar de instellingen voor jouw server
3. Gebruik de tbb2-1.0-complete.sql dump als basis voor de database
4. Ga naar het forum, registreer je account
5. Ga naar het database beheer programma en voeg in de tabel 'tbb_administrators' je nieuwe userID als 'master' toe.
6. Je kunt nu volledig het forum configureren.
