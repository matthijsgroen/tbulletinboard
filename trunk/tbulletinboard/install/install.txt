Installatie
-----------
1. Kopieer de php files naar een omgeving waarin PHP 5 draait.
2. Bewerk de config/settings.php file naar de instellingen voor jouw server
3. Gebruik de tbb2-1.0-complete.sql dump als basis voor de database
4. Ga naar het forum, registreer je account
5. Ga naar het database beheer programma en voeg in de tabel 'tbb_administrators' je nieuwe userID als 'master' toe.
6. Je kunt nu volledig het forum configureren.
7. Zorg dat alle submappen van 'upload' op 777 staan
8. start modules/create.php om alle modules te maken (verschijnen in upload map)
9. Via beheer kunnen de modules worden geupload en worden geinstalleerd.

optioneel:
10. Download de laatste dump van http://s5.travian.com/map.sql

Werken aan modules
------------------
Om te zorgen dat het werken aan modules makkelijker is, zorg dan dat in config/settings.php 
$developmentMode = true;
aangegeven staat. Op deze manier wordt een module die geinstalleerd en geactiveerd is niet uit /upload/modules/ geladen,
maar direct uit /modules/ Op deze manier heb je geen rechten problemen en kun je direct de code veranderingen in de modules
maken en bekijken. om de module weer in te pakken en te downloaden kun je de stap 8 hierboven uitvoeren of naar plugin
beheer gaan en daar de module downloaden (alleen met developmentMode = true!)
